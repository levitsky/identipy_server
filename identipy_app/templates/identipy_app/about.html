Software performs:

  • database search for peptide and protein matching [1];
  • validation of the protein identification results using Scavager algorithm [2];
  • filtering to a given percentage of global false discovery rate (FDR) using target-decoy approach;
  • label-free quantitation using spectral counting (Normalized Spectral Abundance Factor [3]).

This software provides a Web interface for specification of search parameters and inspection of results. IdentiPy search engine supports the following formats:

  • FASTA for protein databases
  • MGF or mzML for MS/MS data
