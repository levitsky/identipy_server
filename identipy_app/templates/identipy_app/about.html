IdentiPy Server, version 1.0, release date 01.01.2017

Software performs:

  • database search for peptide and protein matching with calculation of HyperScore [1] or MorpheusScore [2] for ranking peptide-spectrum matches;
  • filtering to a given percentage of global false discovery rate (FDR) using target-decoy approach [3];
  • validation of the protein identification results using MPscore algorithm [4];
  • label-free quantitation using spectral count based methods (Normalized Spectral Index [5], Exponentially Modified Protein Abundance Index [6], Normalized Spectral Abundance Factor [7]).

This software provides a Web interface for specification of search parameters and inspection of results. IdentiPy search engine supports the following formats:

  • FASTA for protein databases
  • MGF or mzML for MS/MS data

To start your search, please, log in or contact the administrators to create an account.
